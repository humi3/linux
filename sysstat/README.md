# sysstatについて
サーバのリソース情報をバイナリで収集、管理するツールです。  
情報の種類としては、CPUやメモリ使用率、トラフィック量、ディスクI/Oなど様々な情報が取得できます。

## install
```
# yum localinstall http://mirror.centos.org/centos/7/os/x86_64/Packages/sysstat-10.1.5-18.el7.x86_64.rpm
# yum -y install sysstat
```
